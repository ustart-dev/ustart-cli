# ustart-cli

The uStart CLI.

## Installation

```
$ npm install -g ustart-cli
```

## Getting started

First, create a new folder with your project's name:

```
$ mkdir your-awesome-project
$ cd your-awesome-project
```

Then, initialize it with *ustart-cli*:

```
$ ustart init
```

## Usage

```
$ ustart --help
$ ustart --version
$ ustart init
```
